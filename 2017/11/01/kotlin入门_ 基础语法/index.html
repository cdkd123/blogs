1.9.4 使用range

使用in操作符检查x是否在某个范围：

val list = listOf(“a”, “b”, “c”)
​
if (-1 !in 0..list.lastIndex) {
println(“-1 is out of range”)
}
if (list.size !in list.indices) {
println(“list size is out of valid list indices range too”)
}

for (x in 1..5) {
print(x)
}

for (x in 1..10 step 2) {
print(x)
}
for (x in 9 downTo 0 step 3) {
print(x)
}

when {
“element1” in items -> println(“elment is in items”)
“element2” in items -> println(“elment2 is in items”)
}

for(item in items) {
println(“$item”)
}

items.filter(it -> it.key > 5).map(it -> println(it))
