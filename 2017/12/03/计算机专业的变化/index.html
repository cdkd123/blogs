计算机专业的变化

国内计算机专业课程

我是07年上的大学，那时候我们学校（三本）报考的专业，就有“计算机科学与技术”，其实并没有什么概念，就报了。只是感觉和“游戏”，“写代码”，“黑客”这样的词有关系。于是非常兴奋的报了。连学校要教的课程都没有看。

不妨再看看清华大学的计算机(2011年）：

这样看来，我们学校的课程真是弱爆了！！

所有这些专业，其实总结起来，无非是软件，硬件所需知识。国内的产业都需要这些人才。但这些知识其实并不够在社会上找到一份IT行业的好工作！

国外的计算机科学与技术

最近扫了一眼Coursera上的伊利诺伊州立大学的计算机专业，有些惊讶。他们清楚的定义自己是”数据科学领域的计算机科学硕士“专业。他们教授的已经是纯软件，纯数据方向的知识了。

再看看看coursera首页对计算机科学的描述：

· 计算机科学专项课程教授软件工程和设计、算法思维、人机交互、编程语言和计算机历史。本类课程涉及领域比较宽泛，它们将帮助您提高抽象思维能力，系统性地解决问题，提出合理的解决方案。

主要课程列表：

  • Applied Data Science with Python
  • 数据结构与算法
  • 移动和网络开发 - Android应用与开发
  • Design Paterns
  • 响应式网站开发与设计

在人才需求方面，网页设计，移动应用会比较多，游戏开发需求也不少，但，其实说，这些领域最近两年人才快速增长，如果现在入学就学，毕业未必不会人才过剩呢。

所以，要我说，去报“AI”方向吧。AI不会人才过剩？我认为，

  1. 学习成本高，回报也高。并不是两三个月就能学会的。
  2. 非常适合学生在校学习。
  3. 高端AI人才的缺口非常大。（统计数字：）
  4. AI是国家战略发展方向。所以有前途。

但是国内的AI技术，现在不论是媒体，还是科技公司对外宣传，感觉都吹的有点过了。
